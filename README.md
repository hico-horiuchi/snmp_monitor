# Monitor

[![Build Status](https://travis-ci.org/hico-horiuchi/snmp_monitor.svg?branch=master)](https://travis-ci.org/hico-horiuchi/snmp_monitor)

MonitorはSNMPを「見える化」するツールです。  
サーバのCPUやメモリの使用率をグラフ化します。
